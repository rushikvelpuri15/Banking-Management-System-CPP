# Banking-Management-System-CPP
Console-based Banking System Using C++ with OOP and File Handling
Banking Management System (C++)

📌** Project Overview**
This is a console-based Banking Management System developed using C++ implementing Object-Oriented Programming and File Handling concepts.
The system allows users to: 
Open Account
Deposit Amount
Withdraw Amount
Check Balance
Close Account
View All Accounts
All account data is stored in a file (Bank.data) ensuring persistent storage.

🛠 **Concepts Used**
OOP (Classes & Objects)
Constructors
Static Members
STL Map
File Handling
Operator Overloading
Exception Handling

**⚙ How to Run**
Compile using any C++ compiler
Run the program
Use menu to perform operations
